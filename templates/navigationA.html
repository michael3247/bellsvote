For Admin Users

As an Admin, you have special privileges to manage elections, view results, control user accounts, and oversee the voting process.

1. Login

Login using your email:

2. Admin Dashboard

Once logged in, you will be redirected to the Admin Dashboard. The key functionalities here are:

  • Post Creation: Create new elections by selecting a predefined post or providing a custom one. Set the start and end times for the election.
  • Managing Elections: You can view and manage active elections. Admins can start elections, view details, and manage candidates.
  • User Management: Admins can view all accounts through the "Users" section.
  • Results: View results of completed elections via the "Results" link.

3. Active Elections

Admins can view ongoing elections and remove candidates if necessary. The "Remove" button allows you to remove candidates from the election.

4. Account Control

As an Admin, you can:

  • Logout: Click "Logout" to exit your account.
  • Delete Account: Admins can delete their accounts through the "Delete Account" button in the Account section. This action is irreversible.

Common Features for Both Admins and Students

Navigation Bar

The navigation bar provides quick access to key sections of the portal:

  • Dashboard: Admins see their control panel; students view active elections.
  • Active Elections: View and vote in ongoing elections.
  • Results: View results of completed elections.
  • Account: View and manage your account settings.

Time Format for Elections

Elections have a clear start and end time. You can only vote during the active period. Once voting ends, no further votes can be cast.

Important Notes:

  • Election Integrity: Ensure you vote only during the active election period.
  • Voting Restriction: Each user can vote only once per election.
  • Data Privacy: User data, including votes, is secure and kept confidential.
